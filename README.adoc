# Arista EOS Extension Repository

## File Types & Their Work
- **`.py`**: Main logic source.
- **`.pyx`**: Cython performance source (compiles to C).
- **`.pyc`**: Compiled bytecode for fast execution (auto-generated).
- **`.cpy`**: Provisioning templates for ZTP.
- **`.swix`**: The final deployable extension bundle.

## Build Process
1. Package `src/` into an RPM.
2. Run `swix-create -i manifest.yaml output.swix my-tool.rpm`.
3. Verify with `python3 scripts/manage.py`.
